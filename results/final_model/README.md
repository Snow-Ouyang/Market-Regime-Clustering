# Final Model Outputs

This folder contains the curated outputs for the **preferred final specification**:

- 4-state Jump Model
- variables: `growth_pc1`, `inflation_pc1`, `gs10`, `term_spread_10y_1y`, `credit_spread`
- penalty: `p = 1.1`

Key files:

- `regime_labels.csv`: final regime names
- `regime_panel.csv`: monthly state assignments and state probabilities
- `state_moments.csv`: state-level means and variances for the modeled variables
- `regime_characteristics_summary.csv`: compact summary table used in the README
- `asset_performance_by_regime.csv`: cross-asset performance by regime

See `../../figures/final_model/` for the GitHub-facing figures.
