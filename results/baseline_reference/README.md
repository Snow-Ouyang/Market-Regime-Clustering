# Baseline Reference Outputs

This folder contains the curated outputs for the **reference benchmark**:

- 3-state compact Jump Model
- variables: `growth_pc1`, `inflation_pc1`, `gs10`, `term_spread_10y_1y`
- penalty: `p = 0.6`

These files are retained because the baseline is still useful as a parsimonious and stable comparison model, even though it is no longer the preferred final specification.

Key files:

- `penalty_grid_summary.csv`
- `time_stability_summary.csv`
- `state_count_summary.csv`
- `regime_labels.csv`
- `asset_performance_by_regime.csv`

See `../../figures/baseline_reference/` for the benchmark figures referenced in the documentation.
