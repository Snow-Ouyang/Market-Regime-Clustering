# Core Results

These folders contain the main outputs used in the final project narrative:

- `jump_model_penalty_grid/`: final local penalty grid and regime profile summaries
- `jump_model_time_stability/`: trimmed-sample stability analysis
- `jump_model_state_count_stability/`: 2/3/4-state comparison under the final baseline
- `regime_interpretation/`: final regime labels, external validation, and asset mapping
